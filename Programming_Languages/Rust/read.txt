In order to create a new rust-project do:
1. go into the rust-directory
2. mkdir new_proj_name + cd new_proj_name
3. cargo new new_proj_name --bin --> builds the cargo project + main file
4. ./target/debug/new_proj_name --> runs compiled code for new_proj_name
